# ESTATE вЂ“ Luxury Real Estate WordPress Theme

A custom WordPress theme developed for a luxury real estate website. The project focuses on creating a premium user experience inspired by leading luxury property websites in the UK and US. The theme was built from scratch without using page builders, providing complete control over the design, performance, and code structure.

## Overview

This project was created as a custom WordPress theme using PHP, HTML, CSS, JavaScript, and Tailwind CSS. Instead of relying on Elementor or other visual builders, the entire UI was hand-coded to achieve a unique, modern, and high-end design.

The website includes multiple responsive pages, smooth animations, and a luxury-inspired interface suitable for showcasing premium real estate properties.
